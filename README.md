# Exercicio-Comando

Implementação de um pequeno banco de dados de
pessoas operado por linha de comando
• Sintaxe: java BancoPessoas <comando> [<args>]
• Comandos: new <id> <nome>, delete <id>, all, get <id>
• Classe Pessoa: id: int, nome: String. Use um HashMap para
implementar o banco de pessoas, e outro para guardar os
comandos.
